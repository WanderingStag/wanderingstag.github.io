# Sync-InTune

## SYNOPSIS
Will sync device with InTune/MEM.

## SYNTAX

```
Sync-InTune [<CommonParameters>]
```

## DESCRIPTION
Will initiate the sync process with InTune/Microsoft EndPoint Manager to receive new policies and report information.

## EXAMPLES

### EXAMPLE 1
```
Sync-InTune
```
